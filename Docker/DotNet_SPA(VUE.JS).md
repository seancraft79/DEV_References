# DotNet Core SPA 만들기
### 설치할 수 있는 .net core spa template list 조회
```
dotnet new --list
```

### 기타 .net spa template (aurelia, knockout, vue 등) 설치
```
dotnet new --install Microsoft.AspNetCore.spaTemplates::*
```

### Vue project 시작
```
dotnet new vue
```

###  Install npm modules
```
npm install
```

### 소스코드 수정시 바로 적용
```
dotnet watch run
```


### 자동 refresh option
```
HotModuleReplacement = true
```

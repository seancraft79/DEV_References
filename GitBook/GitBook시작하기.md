# GitBook CLI 이용 시작하기

##### Node.js 버전확인
[Node.js](https://nodejs.org)
```
$ node -v
```

##### Install gitbook-cli
```
$ npm install gitbook-cli -g
$ gitbook install
```

##### GitBook 만들기
```
$ gitbook init ./<BookName>
```

##### Local 환경에서 호스팅
```
$ gitbook build
$ gitbook serve   // http://localhost:4000 에서 호스팅
```
